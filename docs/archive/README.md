# Archived Documentation

These Markdown files are historical notes from past one-off fixes and
investigations. They are kept for reference but are not part of the current
operating documentation for the site.

Current, maintained documentation lives in:

- `react-site/README.md` — developer setup, scripts, environment, routing,
  contact form, error reporting, privacy/analytics, and content workflow.
- `sanity/` — Studio schema and configuration.

If anything in these archived files conflicts with `react-site/README.md`,
treat `react-site/README.md` as the source of truth.

## Contents

- `FALLBACK_UPDATE_SUMMARY.md` — notes from an earlier content fallback pass.
- `GOOGLE_RICH_RESULTS_IMPLEMENTATION.md` — history of JSON-LD / rich results
  rollout (the current implementation lives in `src/components/StructuredData.jsx`).
- `LOCALHOST_SANITY_CORS_FIX.md` — one-off CORS troubleshooting for localhost.
- `OPTIMIZATION_REPORT.md` — earlier performance audit findings.
- `SANITY_ERROR_FIX_INSTRUCTIONS.md` — step-by-step notes from a Sanity
  troubleshooting session.
- `SEO_TESTING_GUIDE.md` — historical manual SEO QA checklist.
