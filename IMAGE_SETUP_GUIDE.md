### **Step 2: Download Process**

1. **Right-click on each image** on the original website
2. **Select "Save image as..."**
3. **Save to**: `/Applications/US Mechanical Website/images/`
4. **Use the exact filenames** listed above

### **Step 3: Image Specifications**

#### **Recommended Image Sizes:**
- **Team photos**: 400x400px minimum
- **Project photos**: 800x600px minimum  
- **Hero images**: 1200x800px minimum
- **Logos**: 200x100px minimum

#### **Image Formats:**
- **Photos**: JPG format
- **Logos**: PNG format with transparent background
- **Optimize**: Compress images for web (under 500KB each)

### **Step 5: Image Optimization**

#### **Before Adding Images:**
1. **Resize** images to appropriate dimensions
2. **Compress** images for faster loading
3. **Rename** files to match the required filenames
4. **Test** that images display correctly

#### **Tools for Image Optimization:**
- **Online**: TinyPNG.com, Compressor.io
- **Desktop**: Photoshop, GIMP, Preview (Mac)
- **Command Line**: ImageMagick

### **Step 6: Testing**

After adding images:

1. **Refresh** your browser at `http://localhost:8000`
2. **Check** that all images load properly
3. **Test** on mobile devices
4. **Verify** image alt text is descriptive

### **Step 7: Fallback Images**

If you can't find specific images, the website will still work with placeholder text. The images are referenced but not required for functionality.
